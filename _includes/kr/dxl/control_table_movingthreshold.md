움직임의 유무를 판별하는 기준 속도로 사용됩니다.  
Present Velocity(128)의 절대값이 Moving Threshold(24)보다 크면, 움직임 유무를 나타내는 Moving(122) 이 ‘1’이 되고, 작으면 ‘0’이 됩니다.

|      |         값         |                설명                  |
|:----:|:------------------:|:-----------------------------------:|
| 단위 | 약 0.229 [rev/min] | 속도 관련 Data의 단위는 모두 동일함        |
| 범위 |     0 ~ 1,023      |                  -                  |
