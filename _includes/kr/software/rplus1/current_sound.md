
제어기 내부의 마이크를 통해 현재까지 감지한 소리 횟수를 실시간으로 알 수 있습니다.

##### 파라미터 속성

- 현재 소리감지 횟수는 최대 255까지 셀 수 있습니다.
- 소리가 감지될 때 마다 실시간으로 파라미터 값이 증가합니다.
- 일정 시간 동안 소리가 입력되지 않으면 현재 소리감지 횟수값을 소리감지 횟수 파라미터로 넘겨준 후, 자동으로 0 으로 초기화 됩니다.
